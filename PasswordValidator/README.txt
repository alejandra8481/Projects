Readme File for the Password Checker Program
This program has been authored by Alejandra Palacios for Nova Southeastern University ISEC640 Database Security Course with Dr. Ajoy Kumar.  
This release is dates 6/14/2020.

The program is using the latest version of Python3: https://www.python.org/downloads/release/python-383/ 

The program has been tested on a Windows 10 Operating System for expected functioanlity.

To run the program, open the command line and navigate to the directory where all files have been unzipped. The oldpasswd.txt file must be kept in the same directory as pwvalidator.py.

Run the pogram by running the following command: python pwvalidator.py
